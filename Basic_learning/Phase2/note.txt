จัดกลุ่มข้อมูลพื้นฐาน

List = [] , ข้อมูลต่างชนิดกัน ,แก้ไขสมาชิกข้อมูลได้, ข้อมูลซ้ำกันได้, ซ้ายไปขวา
Tuple = () ,ข้อมูลต่างชนิดกัน ,แก้ไขสมาชิกข้อมูล "ไม่" ได้, ข้อมูลซ้ำกันได้, ซ้ายไปขวา

Set = {} ,ข้อมูลต่างชนิดกัน ,แก้ไขสมาชิกข้อมูลได้, ข้อมูลซ้ำกัน "ไม่" ได้, ลำดับไม่แน่นอน
Dict = {key : value}  ข้อมูลต่างชนิดกัน, แก้ไขสมาชิกข้อมูลได้, ข้อมูลซ้ำกัน "ไม่" ได้, ลำดับไม่แน่นอน (ใช้ key เข้าถึงข้อมูลได้)
